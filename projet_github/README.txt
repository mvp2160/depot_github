OBJECTIFS PEDAGOGIQUES

- comprendre le travail collaboratif avec git et github
- lier un depot git local à un depot distant
- utiliser vim comme editeur principal 
- assurer la traçabilite des contributions individuelles 
- developper l'autonomie dans la recherche technique

DESCRIPTION DU PROJET

- le projet debute par le clonage d'un depot github public fourni par l'enseignant, servant de base de travail. A partir de ce depot, chaque groupe devra creer son popre depot github, qui deviendra le depot officiel du projet.

- les etudiants devrons ensuite lier leur depot git local à ce depot de groupe et y ajouter tous les membres du groupe ainsi que l'enseignant en tant que collaborateurs.

- le travail consistera a modifier les fichiers existants à l'aide de l'editeur vim, à versionner les changements avec git 
et à les synchroniser via github.

- chaque membre du groupe devra effectuer au mininmum 3 pulls distincts et produire des contributions visible et verifiable sur le depot.

ORGANISATION DU TRAVAIL

- le present travail a ete effectue par deux etudiants dans un environnement linux, avec vim comme editeur exclusif.
